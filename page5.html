REFLECTION

This activity involved creating an HTML page to represent my daily routine. It was a practical exercise in structuring content using various HTML elements to clearly present information. I focused on using semantic HTML to organize the different parts of my day, from morning activities to evening tasks. I also experimented with styling the page to make it visually appealing and easy to read. The challenges included deciding on the best way to represent the information (e.g., using lists, headings, and paragraphs) and ensuring the page was responsive for different screen sizes. This activity reinforced my understanding of HTML structure and the importance of clear, semantic markup for creating accessible and user-friendly web pages.
