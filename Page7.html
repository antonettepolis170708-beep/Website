REFLECTION

Before redoing exercises 2-6 in three CSS versions, I just picked whatever was quickest. Now I get why each matters. Inline is fast like making my exercise 2 heading blue in one tag but add more styles and the HTML gets messy as hell. Cant use it for whole sections no more. Internal is way cleaner all styles in the head, so I could style my exercise 3 nav links all at once. But copying it to my exercise 5 gallery page was a pain when I had to update colors twice. Good for single pages only. External seemed like extra work at first, but wow one file for all activities. Fix the exercise 5 grid once, it updates everywhere. Change the whole sites font in one line. Def my go-to now. I was winging it before now I know each has a job. Still mess up syntax sometimes, but Im way more confident picking the right one.
