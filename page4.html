REFLECTION

This activity was a deep dive into the core elements of HTML: lists, images, video, and audio. I learned how to structure content effectively using different types of lists (ordered, unordered, and description lists), and how to embed various media types to enhance user experience. Working with images taught me about optimization and responsive design, while integrating video and audio required understanding different formats and embedding techniques. This exercise not only solidified my understanding of HTML but also highlighted the importance of semantic markup and accessibility in web design. The challenges I faced included ensuring cross-browser compatibility and optimizing media for different devices, which pushed me to research and implement best practices. Overall, this activity was crucial in building a solid foundation for creating engaging and accessible web content
